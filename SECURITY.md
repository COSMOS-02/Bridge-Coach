# Security Policy

## Bridge Coach Security

At Bridge Coach, protecting user data and maintaining platform security are top priorities. We appreciate responsible disclosure of any security vulnerabilities. We are committed to continuously improving our security practices to protect user information and platform integrity.

---

## Supported Versions

| Version | Supported |
|----------|-----------|
| Current production | ✅ |
| Previous releases | ❌ |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Email:
bridgecoach.contact@gmail.com

Include:

- Description of the issue
- Steps to reproduce
- Screenshots (if applicable)
- Potential impact

Please do not publicly disclose the vulnerability until it has been reviewed and addressed.

---

## Response Timeline

- Initial acknowledgment: Within 48 hours
- Investigation begins: Within 5 business days
- Status updates provided during the investigation
- Critical vulnerabilities are prioritized and patched as quickly as possible.

---

## Scope

This policy covers:

- Bridge Coach Website
- AI Assistant
- Authentication System
- APIs
- User Accounts
- Dashboard

---

## Responsible Disclosure 
We kindly request that security researchers avoid accessing, modifying, or deleting user data during testing. Please act in good faith and provide sufficient information for us to reproduce and resolve the issue.

## Thank You

We appreciate security researchers and responsible disclosure that helps keep Bridge Coach safe for everyone.
